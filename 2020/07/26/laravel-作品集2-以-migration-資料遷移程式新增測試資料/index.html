[laravel][作品集2] 以 migration 資料遷移程式新增測試資料

規劃設計一個能有左選單樹進行分類的作品集介面，如下圖:

在這章節中，先新增測試資料，方便後續撰寫程式時顯示。 一樣使用 migration資料遷移程式對作品表與作品分類表插入資料。

一樣與 新增表格一樣，下指令新增 migration 資料遷移程式:

php artisan make:migration insert_portfolio_category_table

成功新增 migration 資料遷移程式:

依照如上圖的樹狀選單，設計出階層分類的作品集資料:

  • 總目錄
    • 大學作品
      • 專題作品
      • 研究所開發系統/論文
    • 自行開發系統
      • 網頁
      • 手機 app

在 insert_portfolio_category_table 分類的 up() 函式撰寫程式內容，
一樣在 up( ) 執行 插入資料的語法:

DB::table('[資料表名稱]') -> insert( array ( ..., ... ) );

依照如上設計，第一層分類為 大學作品，自行開發系統，第二層分類是專題作品、網頁… 以此類推。
